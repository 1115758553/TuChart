TuChart
=================
TuChart是一个基于Echarts和Tushare的中国股票可视化应用。

Tuchart目前支持的数据类型有：
*未复权日线，周线，月线，15分钟线,30分钟线*
*复权日线，周线，月线，5分钟线，15分钟线,30分钟线*
